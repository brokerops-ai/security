At a glance

  • Privacy-by-default: names masked; policy numbers HMAC-SHA256.
  • No data retained by default; files processed in-memory.
  • Encryption: TLS in transit; encrypted disks at rest.
  • Optional, explicit toggles: Azure OCR (PDFs), OpenAI (aggregates only).

Data flow

  1. Upload CSV/XLSX/PDF over HTTPS.
  2. Process & mask in memory. Outputs: clean CSV/XLSX, issues, HTML report, optional ZIP.
  3. Optional: PDFs sent to Azure OCR; aggregates (counts/sums/dates) sent to OpenAI for a 5-bullet summary.
  4. Minimal diagnostics; filenames hashed/redacted.

Subprocessors

  • Microsoft Azure Document Intelligence — OCR for PDFs (opt-in only).
  • OpenAI (Responses API) — aggregate metrics only; no rows/PII (opt-in only).
